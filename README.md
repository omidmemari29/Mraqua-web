# MrAqua Web (PWA)

نسخه‌ی وب نصب‌شونده (Progressive Web App) از MrAqua — روی آیفون و اندروید، از طریق مرورگر
نصب می‌شه و مثل یه اپ واقعی روی صفحه‌ی اصلی می‌شینه. کاملاً رایگان، بدون اپ استور.

## چی الان کار می‌کنه؟

- صفحه‌ی اتصال (وای‌فای + حالت نمایشی/Demo برای تست بدون سخت‌افزار)
- صفحه‌ی خانه (وضعیت دما + خروجی‌ها با همون تایل‌های درخشان اپ اصلی)
- صفحه‌ی سنسورها (واحد دما، دمای فعلی سه سنسور)
- صفحه‌ی تنظیمات (زبان، پوسته، فرمت ساعت، اندازه‌ی رابط، همگام‌سازی ساعت)
- نصب‌پذیری کامل (manifest.json + service worker + آیکون‌ها)

## چی هنوز جا مونده؟ (بخش «بیشتر» فعلاً Placeholder ـه)

خروجی‌ها (کنترل کامل)، دوزینگ، ATO، نور، تغذیه، پارامترهای آب، دفترچه، هشدارها — این‌ها
توی اپ اندروید کامل‌ان ولی اینجا هنوز پیاده نشدن. ساختار کد (js/app.js) طوریه که اضافه‌کردن
هرکدوم فقط نیاز به یه تابع render + bind جدید داره، دقیقاً مثل الگوی renderSensors/renderSettings.

## ⚠️ مهم‌ترین پیش‌نیاز: پل WebSocket روی فرمور

مرورگرها (چه آیفون چه اندروید) نمی‌تونن مثل اپ اندرویدی یه Socket خام TCP باز کنن.
باید روی ESP32/ESP8266 یه سرور WebSocket ساده اضافه بشه (مثلاً با کتابخونه‌ی
`ESPAsyncWebServer` + `AsyncWebSocket`) که همون خط‌های متنی پروتکل فعلی
(`STATUS?`، `OUT:...`، `DOSE:...` و غیره — نگاه کن به `js/protocol.js`) رو از روی
WebSocket بخونه/بفرسته به‌جای Socket خام. تا وقتی این پل ساخته نشده، فقط از
**«حالت نمایشی»** توی صفحه‌ی اتصال می‌تونی برای تست UI استفاده کنی.

## تست محلی (قبل از دیپلوی)

```bash
cd mraqua-web
python3 -m http.server 8000
# بعد مرورگر رو باز کن: http://localhost:8000
```

## دیپلوی رایگان روی Cloudflare Pages

1. یه ریپازیتوری گیت‌هاب بساز و کل پوشه‌ی `mraqua-web` رو توش push کن.
2. برو به [pages.cloudflare.com](https://pages.cloudflare.com) و با گیت‌هابت وارد شو.
3. «Create a project» → «Connect to Git» → ریپازیتوریت رو انتخاب کن.
4. تنظیمات build: چون این یه سایت استاتیک ساده‌ست (بدون فریم‌ورک)، فیلد
   **Build command** رو خالی بذار و **Build output directory** رو `/` بذار.
5. Deploy بزن — چند ثانیه بعد یه آدرس مثل `mraqua.pages.dev` می‌گیری (رایگان، HTTPS خودکار).
6. اختیاری: توی تنظیمات پروژه یه دامنه‌ی اختصاصی (که خودت جدا خریدی) هم می‌تونی وصل کنی.

## نصب روی آیفون

بعد از دیپلوی، آدرس رو توی **Safari** (حتماً Safari، نه کروم) باز کن → دکمه‌ی Share
(مربع با فلش) → **Add to Home Screen**. آیکون MrAqua میفته روی صفحه‌ی اصلی و با زدنش
بدون نوار آدرس مرورگر باز می‌شه، دقیقاً مثل یه اپ نصب‌شده.

## ساختار فایل‌ها

```
mraqua-web/
  index.html          ← پوسته‌ی اصلی PWA
  manifest.json        ← تنظیمات نصب (آیکون، اسم، رنگ)
  service-worker.js    ← کش کردن فایل‌ها برای باز شدن آفلاین
  css/styles.css       ← سیستم طراحی (رنگ‌ها، تایل‌های درخشان، کارت‌ها)
  js/protocol.js       ← ترجمه‌ی دقیق CommandProtocol.kt به جاوااسکریپت
  js/store.js          ← اتصال WebSocket + مدیریت state (معادل AquariumViewModel)
  js/app.js            ← روتر بین صفحات + رندر هر صفحه
  icons/               ← آیکون‌های اپ (192/512 و نسخه‌ی maskable)
```
